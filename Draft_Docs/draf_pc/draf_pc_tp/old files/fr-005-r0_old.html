Introduction

This document details the test steps I2, which are performed on the fault recorder type 16132 manufacturer Hathaway, on a monthly basis M1.

Safety Precautions
  • Work order should be issued,
  • The DFR should be in service,
  • Appropriate personnel protective equipment must be used.

Tools and Equipment

  • Normal multimeter or digital Multimeter,
  • Complete set of screwdrivers.
  • Blower, musk.

Work to be Carried Out

Clean both inside / outside of the fault recorder panel from dust using the blower.

Checking the Operation Lamps

  1. Check the lamps (6, 7, 8) should be OFF.

  2. Check the alarm lamps no. (1, 2, 3) should be OFF.

Checking the Printer & DC/DC Converter

  1. Check the lamps no. (1, 2, 4) should be ON, and lamp no. 3 OFF

  2. Check visually the DC / DC converter
