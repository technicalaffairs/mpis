Introduction

 This procedure is used to perform off load relay test.

Safety Precautions

  • A work permit must be issued and the qualified maintenance staff should carry the suitable safety category,

  • Safety fence (rope) with caution marks surrounding the work area,

  • Line isolation is required.

  • Make sure CT secondary circuits are shorted before isolating the relay current circuits for test current injection.

  • Switch off DC power supply before inserting or withdrawing any card from the relay.

 Tools and Equipment

  • AEG Relay test set 74-2

  • Voltmeter

  • Clip - on ammeter

  • Test leads

Work to be Carried Out

Steps to prepare for relay testing

  1. Review technical reference material  Manufacturer's manual

  2. Obtain copy of relay setting from field maintenance file

  3. Obtain outage for line protected by this relay

  4. Check that the bus and line disconnect switches are opened and tagged

  5. Inspect the CT secondary circuits the VT Miniature Circuit Breakers (MCB`S), and all cabling supplying the relay

  6. Before inserting the test plug handle into the relay check from the station drawing that the test plug will isolate
          * Breaker trip coils and transfer trip scheme
          * Breaker failure scheme associated with this relay
          * Disturbance alarms and fault recorder associated with this relay
          * Auto-recluse associated with this relay
             If the test plug does not isolated any one of above items, you have to isolate it.

Testing the relay, relay scheme, and protection outputs 

  1. Carefully follow this test procedure .    

  2. Verify the test results match the relay setting-If not consult technical services

  3. Perform checks of the acceleration of the relay, breaker failure and recluse, separate technical procedures should be used to test each of this system

  4. Test trip all breakers associated with the relay.

Preparation

  1. Turn the switch on the front of the distance relay on (PSN 1000) card clockwise direction 1/4 turn, this will move the switch from the service position (Horizontal) to the out of service position (Vertical), at this moment the red lamp on the same card will light up.

  2. Connect the plug gable interconnected cable between the two units.

  3. Insert the test plug handle into the relay.

  4. Arrange the switches on the relay test set 74-2 as follow Unit Test Set . 

On Current Unit Test Set        Figure 1

  1. ( b1 ) Main switch, power supply, ON-OFF, Set it on (Aus) position      

  2. ( b2 ) Push button test, (off) 

  3. ( b3 ) Push button test, (on)

  4. ( b4 ) Push button “Automatic Disconnection” (Via the contacts of the protection relay on / trip ), on (off) position. Set it on (off) upper position

  5. ( b5 ) Resetting or cancellation resp. of U > cancel.

  6. ( b6 ) Push button bias voltage ( ON / OFF ).

  7. ( b7 ) Change over of the external ( trip command ).

  8. ( b8 &b10 ) Selecting switches for the current ranges, should be set to value more than maximum current that could be supplied to the relay. From position of its we find the value of  factor m.

  9. ( b9 ) Selecting switch for the ranges of the rated current In = 1 Amp. or 5 Amp. set it according to the relay rated current.

  10. ( b18&b19 ) Voltage Selector switches, to choose the type of the fault during the test

  11. ( b20 ) Test current selector switch, to choose the type of the fault during the test

  12. ( b22 ) Un (Distance relay type SD 324 a -mod).

  13. ( m1 ) Current regulating knob to adjust the current value, Rotate the adjustment knob m1 to the minimum current level Zero by rotating it to the fully counterclockwise (ccw) position.

  14. ( g1 ) Ammeter

 On Voltage Unit Test Set         Figure 2

  1. ( b11 ) Plug for the voltage tap, resistance vector. To adjust the value of cos Ø 
    (Ø is the relay angle ) from schedule, we find the value of cos Ø

  2. ( b12 ) Plug for the voltage tap, resistance vector. To adjust the value of sin Ø
    (Ø is the relay angle), from schedule, we find the value of sin Ø

  3. ( b13 ) Quadrant selector switch, and set it on position according to C.T. earthing.
    Position 2 or 3 if the relay plug pin at bus bar side (SS). Set it on position 2
    Position 1 or 4 if the relay plug pin at bus bar side (LS). Set it on position 1

  4. ( b14 ) Mode-of-test switch Z-measuring. U/I-measuring. Set it on (Z) position

  5. ( b15&b16&b17 ) Adjusting switches for the impedance. Set it on (zero) Position %- selector switch (AZ). Set it on position

  6. ( b21 ) Change over switch of the angles for the phase voltage of direction measuring unit, on (zero degree) position

  7. ( b23 ) Change - over switch to select the kind of C.T. earthing. Set it on position according to C.T. earthing

  8. ( b24 ) Voltage regulating knob to adjust the voltage value. Rotate the adjustment knob m3 to the minimum voltage level (zero) by turning it to the fully counterclockwise 

  9. ( m3 )(ccw) position

  10. ( g3 ) Voltmeter

  11. Set the two MCB`s (on current unit test set) on (off) position

  12. The AEG test set can now be energized by plugging the (Main Input) cord into a normal 220 VAC wall outlet.

Checking the pick up value of the Phase to Ground Measuring Elements

  1. Calculate the reselected value Zv to be adjusted to switches b15, b16 and b17 corresponding to the relay setting of the first time step for phase to ground fault where
                    Zv = m. ZL
                    m  = factor from the table (on current unit test set)
                    ZL = protection relay setting (line impedance) for phase to ground fault

  2. Turn the switches b15, b16 and b17 to agree with the zone one setting (Zv calculated).

  3. Select single phase to ground fault by switches b18 and b19 to (LE) position and b20 to (RM) position.

  4. Set the two MCB`s (on current unit test set) to (on) position
    Note : h1 at the same test set will be white color

  5. Turn switch b1 to (Ein) position

  6. Start the test by pushing down the push button b3

  7. Inject current by rotating the knob m1 in clockwise direction until the current exceeds the value of the low-set over current starting If element on the distance relay

  8. Push the push- button b2 to stop the injected current

  9. Push the push- button b4 

  10. Push the push- button b3 and observe that the injected current will stop once the relay has picked up  

  11. Observe that the red indication AR, IM, Nv, T1, Aus R, Aus S, Aus T and general trip (Aus R, Aus S, Aus T) on the relay will light up.

  12. Change the value of zone one impedance which set on the tester by switches b15, b16 and b17 starting by switch b17 starting up or down until the relay operate in second step
    (the red indication T2 on the relay will light up with the last indications)
    The second step time will be recorded on the screen of the timer unit (on current unit test set)

  13. Record this pick up ohm point on test sheet (1)

  14. Repeat the above test for all phases (RM, SM, TM)

       * Repeat the above test, once again but this time, calculate Zv corresponding to the relay setting of the second zone for phase to ground fault and set it on the tester. After that change it up or down until the relay operate in third step (The red indication lamp T3 on the relay will light up with the last indications).

    The third step time will be recorded on the timer unit
       * Repeat the above test once again but this time, calculate Zv corresponding to the relay setting of the third zone for phase to ground fault and set it on the tester. After that change its up or down until the relay operate in fourth step
            (The red indication T4 on the relay will light up with the last indication).
            The fourth step time will be recorded on the timer unit

Checking the Pick up Value of the Phase to Phase Measuring Elements

  1. Calculate the reselected value Zv to be adjusted to switches b15, b16 and b17 corresponding to the relay setting of the first time step for phase to phase fault 
    where Zv = m. ZL

  2. Turn the switches b15,b16 and b17 to agree with the zone one setting (Zv calculated).

  3. Select phase to phase fault by switches b18 and b19 to (LL) position and b20 to (RS) position.

  4. Push the push button b3 and observe that the injected current will stop once the relay has picked up

  5. Observe that the red indications AR , NV , AS , NV , TI , Aus R , Aus S, Aus T and general trip (AusR,Aus S,Aus T) on distance relay will light up

  6. Change the value of zone one impedance which set on the tester until the relay operate in second step
    (The red indication T2 on the relay will light up with the last indications)
    The second step time will be recorded on the timer unit  

  7. Record this pick up ohm point on test sheet 1

  8. Repeat the above test for all phases (RS, ST, TR)
      * Repeat the above test once again but this time, calculate Zv corresponding to the  relay setting of the second zone for phase to phase fault and set it on the tester. After that change it until  the relay operate in third step  
    (The red indication  T3 on the relay will light up with the last indication).
    The third step time will be recorded on the timer unit

     * Repeat the above test once again but this time, calculate Zv corresponding to the relay setting of the third zone for phase to phase fault and set it on the tester. After that change it until the relay operate in fourth step 
    (The red indication T4 on the relay will light up with the last indication).
    The fourth step time will be recorded on the timer unit

NOTE  : Many different pick up points for zone reaches of the relay could be obtain by repeating the above steps at different phase angle by changing switches b11, b12, b13 and b23 

Measuring the Operating Times

  1. Set the line impedance Zv on tester to a value which equals half the normal pick up value 
    in the first time step for phase to earth This will allow a good operation of zone one of the relay

  2. Select single phase to ground fault by switch b18, b19  to (LE) position and b20  the to (RM) position.

  3. Push the push button b3 The timer will stop once the relay has picked up

  4. Record the zone one tripping time which recorded on the timer on test sheet 1.

  5. Repeat the above test to measure the zone two tripping time. Once again but this time set Zv on tester by switches b15, b16 and b17 to a value which half the normal pick up value in second time step for phase to earth.

  6. Repeat the above step to measure the zone three tripping time. Once again but this time set Zv on tester by switch b15, b16 and b17 to a value which equals half the normal pick up value in third time step for phase to earth.

  7. Repeat the above test but for phase to phase fault.
    DO NOT FORGET TO CHANGE
            * Step no 1 to (Set the value of impedance which set on switches b15, b16, and b17 to a value which equals 
               half the normal pick up value of first time step impedance for phase to phase).
            * Step no 2 to (Select the switches b18 and b19 to (LL) position and b20 to (RS) position)

Testing the Tripping Circuit For the Associated Circuit Breakers

  1. Rotate the knobs m1 and m3 to minimum current and voltage levels (zero)   
  2. Push the push- button b4 to off (upper position).
  3. Turn switch b1 to (Aus) position.
  4. Set the two MCB’s to off position.
  5. Draw out the (Main Input) cord from 220 vac wall outlet.
  6. Draw out the test plug of the test set 74 -2 from distance relay.
  7. Turn the switch on the front of the distance relay on ( PSN1000 ) card counterclockwise direction 1/4 turn to put the distance relay in service position, at this moment the red lamp on the same card will go off.
  8. Press the push button (PA) simultaneously with the test button PR, PS, PT, or PM. The tripping signal for the circuit breaker will be performed during this test.

Return Relay and Associated Equipment to Service

  1. Return relay to service by withdrawing the test plug handle from the relay.
  2. If any equipment associated with the relay has been isolated without the test plug, it should be returned to service by the following restoration steps
          * Breaker trip coils and transfer trip scheme.
          * Breaker failure scheme associated with this relay.
          * Disturbance alarms and fault recorder associated with this relay.
          * Auto-reclosure associated with this relay.

Request Power Line be returned to Service

  1. Take crossed readings of all AC currents and voltages supplied to the relay and compare them to those for normal load conditions.
  2. Check that readings agree with station meters (MW/ MVAR/Power Factor/Amperes/voltages)
  3. Place a copy of the test result in the field maintenance file.
  4. Look at the panel nameplate and check that this  system designation nameplate agrees with the line that has been taken out of service for maintenance
  5. Check there is no A.C current from C.T`s by using a clip on ammeter

  6. Check there is no A.C voltage from V.T`s by using a voltmeter
