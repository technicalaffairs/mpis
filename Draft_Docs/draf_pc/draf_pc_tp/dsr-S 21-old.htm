  • Safety rope with caution marks surrounding the work area,

Positive sequence calculation for 3 phases unbalanced voltages

V1 = 1/3 (Va + a vb + a2 VC )            Va = reference = 0º         Vb / Va:= 240º      Vc / Va: = 120º

V1 = 1/3 [(Va + Vb(-0.5 + j0.866) + Vc (-0.5-j0.866)]

 

Negative sequence calculation for 3 phases unbalanced voltages

V2 = 1/3 (Va + a2 vb + a2 VC )            Va = reference = 0º         Vb / Va:= 120º      Vc / Va: = 120º

V2 = 1/3 [(Va + Vb(-0.5 + j0.866) + Vc (-0.5-j0.866)]

 

Same formula apply for currents
