Introduction

This document lists the equipment Inspection checklist I2, which must be performed for Becker compressor type SV 250 max pressure/150 discharge every four months as indicated by M4 in the title block.

Safety Precautions
  • Work order must be issued.
  • Use standard personal safety equipment, safety helmet, and safety glasses.

Tools and Equipment

  • Personal safety equipment, Helmet, and safety glasses

Work to be Carried Out

  1. Check air filter cartridge, if it is dirty, blow the cartridge out with compressed air from the inside outwards, if paper elements in cartridge are damaged, install a new filter cartridge.

  2. Ensure that filter cartridge is properly seated when Hitler housing is re-assembled.

  3. Check oil level in compressor crack case, but only with the compressor at rest. The oil level should never fall below the sight- glass.

  4. Check for any unusual noises or vibrations.

  5. Check for air leaks in piping and pipe connections.

  6. Read and record gauge pressures and compressor running time indicator.

  7. Clean compressor room floor.
